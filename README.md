# 🚀 LaunchMind AI

> An AI-powered startup launchpad designed to help founders validate, plan, and build their startup ideas seamlessly.

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), React, Tailwind CSS, TypeScript
- **Backend:** FastAPI, Python, Pydantic, SQLAlchemy
- **Deployment:** Vercel / Render
